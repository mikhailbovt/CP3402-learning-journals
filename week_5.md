# CP3402 Learning Journal - Mikhail Bovt

## Week 5

## Learning Activities & Resources

This week, I learned how WordPress child themes work and why they are the recommended way to customize existing themes. I created a child theme, explored the purpose of `style.css` and `functions.php`, and practiced making visual customizations without modifying the parent theme. I also reviewed WordPress development practices, coding standards, and the use of browser developer tools for testing CSS changes.

## Estimated Hours of Explicit Learning Activity

5 hours

## Content Insights

The main insight this week was that child themes provide a safe and maintainable way to customize WordPress websites. They allow developers to extend existing themes while preserving compatibility with future updates. I also learned the importance of following coding standards and using proper development workflows when working with WordPress projects.

## Career/Employability/Learning Insights

This week's practical work demonstrated how professional developers maintain and customize large projects without changing core files. Learning to use child themes, version control, and development tools reflects industry best practices and reinforces the importance of writing maintainable, well-structured code.
